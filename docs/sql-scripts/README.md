# Scripts SQL

Ce dossier contient les scripts SQL utilitaires pour la gestion de la base de données Neon.

## Scripts disponibles

### CREATE_USER_NEON.sql
Script pour créer un nouvel utilisateur dans la base de données Neon PostgreSQL.

### UPDATE_USER_NEON.sql
Script pour mettre à jour les permissions d'un utilisateur existant dans Neon.

## Utilisation

Ces scripts sont à exécuter directement dans l'interface Neon ou via psql avec la connexion appropriée.

**Note** : Ces scripts sont des utilitaires de maintenance et ne doivent être exécutés que par des administrateurs de base de données.
