# Vidéos du site

## waveform-gradient.mp4

Cette vidéo contient l'animation du gradient horizontal pour le waveform du hero.

### Spécifications requises :

- **Résolution** : 800x64px (ou 1600x128px pour retina)
- **Durée** : 2-3 secondes (boucle fluide)
- **Frame rate** : 60fps
- **Format** : MP4 (H.264)
- **Taille** : < 500KB
- **Couleurs** :
  - Purple: `#a855f7` (rgb(168, 85, 247))
  - Blue: `#3b82f6` (rgb(59, 130, 246))

### Instructions de création

Voir `/docs/WAVEFORM_VIDEO_CREATION.md` pour les instructions détaillées.

### Fallback

Si cette vidéo n'est pas disponible, le site utilisera automatiquement une animation CSS comme fallback.
