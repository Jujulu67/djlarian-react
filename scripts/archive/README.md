# Scripts Archivés

Ce dossier contient les scripts qui ne sont plus utilisés dans le projet mais qui sont conservés pour référence historique.

## Scripts archivés

- `clean-events.ts` - Script de nettoyage des événements (peut être réutilisé si nécessaire)
- `cleanup_images.js` - Script de nettoyage des images orphelines
- `fix-image-mapping.mjs` - Script de correction du mapping des images
- `fix-missing-images.mjs` - Script de correction des images manquantes
- `map-existing-images.mjs` - Script de mapping des images existantes
- `restore-from-backup.sh` - Script de restauration depuis backup

## Note

Ces scripts ne sont plus référencés dans `package.json` et peuvent être supprimés définitivement si vous êtes sûr qu'ils ne seront plus nécessaires.
